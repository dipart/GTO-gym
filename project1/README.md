# GTO Trainer & Manager v3.0

## 🎯 项目简介

GTO Trainer & Manager 是一个基于Web的扑克博弈论最优（GTO）策略训练器。它提供了一个可视化的13x13手牌网格界面，帮助扑克玩家学习和练习不同情况下的最优加注范围。

## ✨ 核心功能

### 1. 手牌范围可视化
- **13x13网格**: 显示所有169种可能的德州扑克手牌组合
- **颜色编码**: 直观显示不同的行动选择（加注-红色、跟注-绿色、混合-黄色、弃牌-灰色）
- **对子高亮**: 口袋对子特殊显示，便于识别

### 2. GTO算法对比
- **实时建议**: 基于位置和筹码深度的GTO范围计算
- **训练模式**: 显示你的选择与GTO建议的差异
  - ✅ **绿色小勾**: 选择与GTO建议一致
  - ⚠️ **绿色圆点**: GTO建议但你未选择（漏选）
  - ❌ **红色叉号**: 你选择了但GTO不建议（多选）

### 3. 多场景支持
- **游戏模式**: MTT锦标赛、现金局
- **人数设置**: 9人桌、8人桌、6人桌、单挑
- **位置选择**: UTG、UTG+1、LJ、HJ、CO、BTN、SB、BB
- **筹码深度**: 10BB-100BB（MTT）、100BB-200BB（现金局）
- **行动类型**: 率先加注、面对3-bet、盲注防守

### 4. 数据管理
- **本地存储**: 自动保存你的训练配置
- **档案库**: 管理和加载已保存的配置
- **导出/导入**: 支持JSON格式的数据备份和分享

## 🚀 快速开始

### 基本使用
1. 打开 `index.html` 文件
2. 选择游戏模式、人数、位置、筹码深度和行动类型
3. 点击手牌网格设置你的范围
4. 开启"GTO标准答案"查看建议对比
5. 保存当前配置到档案库

### 手牌状态切换
- **单击**: 循环切换状态（弃牌→加注→跟注→混合→弃牌）
- **颜色说明**:
  - 🔴 红色: 加注（Raise）
  - 🟢 绿色: 跟注（Call）
  - 🟡 黄色: 混合策略（Mix）
  - ⚫ 灰色: 弃牌（Fold）

### GTO对比功能
1. 确保选择了"率先加注"行动类型
2. 开启右上角的"GTO标准答案"开关
3. 网格会显示GTO建议与你的选择的对比
4. 根据提示调整你的范围

## 🎮 使用技巧

### 训练建议
1. **循序渐进**: 从简单的位置和筹码深度开始
2. **重点练习**: 重点关注常见位置和筹码深度
3. **对比学习**: 经常使用GTO对比功能
4. **定期复习**: 定期回顾和练习已保存的配置

### 高级用法
- **数据分析**: 导出数据进行分析和学习进度跟踪
- **配置分享**: 与其他玩家分享你的配置
- **多设备同步**: 通过导入导出在不同设备间同步数据

## 🔧 技术特性

- **纯前端实现**: 无需服务器，可直接在浏览器中运行
- **本地存储**: 使用localStorage保存数据
- **响应式设计**: 适配不同屏幕尺寸
- **现代化界面**: 暗黑主题，专业视觉效果

## 📝 注意事项

1. **浏览器兼容性**: 建议使用现代浏览器（Chrome、Firefox、Safari）
2. **数据安全**: 数据存储在浏览器本地，清除浏览器数据会丢失
3. **定期备份**: 建议定期导出重要配置进行备份
4. **GTO算法**: 本工具的GTO算法为简化版本，主要用于训练参考
